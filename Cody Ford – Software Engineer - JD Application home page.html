Creating the Scene

First I got a skid steer loader mesh from an online vendor called TurboSquid.  Then I  exported the mesh from 3D Studio Max  as a FBX file in order to import it into blender.  Afterwords I created textures for the mesh to achieve the desired color and metal look. 

Using Blenders Texture Node Editor I changed the skid steer’s back window to resemble glass. 

I then found and imported a roundabout mesh for the scene, and deleted some arrow signs that were going around the roundabout. To make the roads that connected to the roundabout I selected the vertices of the roundabout and extruded out new planes.

Rigging & Animating

Next I created some drivers for the SSL’s wheels so that they would spin when the SSL is moving.  I then changed the parents of each object that made up the SSL. This made it so that moving the arms would move the objects that are attached to the arm as well. After everything was moving correctly I started to create the animation.   

This meant adjusting the meshes until they were in the right position and then saving their location, rotation and scale as a keyframe.  After creating all of the keyframes and adjusting where they were in the timeline I rendered out each frame using Blenders Cycles engine. Finally I imported them into Adobe Premiere to combine them into the final animation.

OpenGL Real Time World

The Textures

A real time render of the world that combines multiple textures by modifying the OpenGL pipeline and making my own shades. The render responds to various user inputs.

The world is rendered with a combination of all of these textures.

  • Daytime Texture – The normal fully colored world, for when the sun is shining on the world. 
  • Nighttime Texture – The world is mostly dark with lights where there is civilization.
  • Height Map  – Used to make the world more three dimensional. Used instead of a bump map. 
  •  Specular Map – Used to add a specular highlight to the ocean when the sun moves across it. 
  • Clouds Texture – This texture is rotated each frame before being combined with the others. This makes the clouds rotate interdependently  from the rest of the textures. 
Some of the OpenGL shader code for combining all of these textures into one texture that is added to the world sphere. 

 

The Controls

The user can interact with the world by pressing any of these inputs.

  • Arrow Keys – Rotate the world where it is.
  • WASD – Move the world around inside the 3D environment.  
  • e – Move the camera to the center of the inside of the sphere. 
  • q  – Move the camera back to its starting point.
  • t – Toggle the earths rotation on and off.
  • r – Toggle the suns rotation around the earth on and off. 
  • f – Place a light source behind the camera so that the earth is always lit up. 
  • 8 – Shrink the size of the earth.
  • 9 – Increase the size of the earth.
  • 0 – Set the earth back to it’s default size.
